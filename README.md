# Device Tree for Realme C25/C25s/Narzo 50A

The Realme C25/C25s/Narzo 50A are budget smartphones from Realme. These are announced and released in March, June and September 2021.

## C25 Specifications
Basic   | Spec Sheet
-------:|:-------------------------
CPU     | Octa-core (2x2.0 GHz Cortex-A75 & 6x1.7 GHz Cortex-A55)
Chipset | MediaTek Helio G70 (12 nm)
GPU     | Mali-G52 MC2
Memory  | 4 GB
Shipped Android Version | Android 11, realme UI 2.0 (Upgradable to Android 13, realme UI 4.0)
Storage | 64/128 GB (eMMC type)
MicroSD | Up to 512 GB 
Battery | Li-Po 6000 mAh, non-removable
Dimensions | 164.4 x 75 x 9 mm (6.47 x 2.95 x 0.35 in)
Display | 720 x 1600 pixels, 6.50" IPS LCD, 20:9 ratio (~270 ppi density)
Rear Camera  | Triple : 48MP(Global)/13 MP(India); 2MP(depth); 2MP(Macro)
Front Camera | Single: 8 MP
Release Month | 2021, March 27
